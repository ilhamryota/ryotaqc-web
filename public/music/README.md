# Music Files Directory

This directory contains background music for the RyotaQC website.

## Required Files

The music player expects the following files:

1. **ryotaqc.mp3** - Main theme song
2. **qcsongs.mp3** - QC vibes background music  
3. **ryotaqccountry.mp3** - Country edition theme

## Adding Music Files

To enable music playback:

1. Add your MP3 files to this directory
2. Name them exactly as listed above
3. Recommended: Use MP3 format, 128-320kbps quality
4. File size: Keep under 10MB per file for optimal loading

## Music Player Features

The music player includes:
- ✅ Autoplay on page load
- ✅ Play/Pause controls
- ✅ Next/Previous track navigation
- ✅ Volume control (mute/unmute)
- ✅ Seekable progress bar
- ✅ Playlist view

## Note

If music files are not present, the player will show but playback will fail silently.
Add your MP3 files here to enable music playback on the website.
